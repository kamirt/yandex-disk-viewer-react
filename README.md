# yandex-disk-viewer

> a simple viewer for yandex disk

## Инструкция по запуску приложения

Переход к root директории по клику на "логотипе"

``` bash
# Установка зависимостей
npm install

# Инициализация базы данных
npm run createtables

# Запуск сервера node.js
node app.js

# Запуск клиента
npm run start
```
